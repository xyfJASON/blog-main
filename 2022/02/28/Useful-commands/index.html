Useful commands

本文收集一些有用的命令。

ssh & scp

包含跳板机：

wake on lan

screen

创建新的 screen 会话

显示当前所有的 screen 会话

手动 attach 指定 screen 会话，断线之后重连就用这个

手动 detach 指定 screen 会话，被 detach 后会话仍然在后台运行，可通过 screen -r 重新绑定

Ctrl+a 系列：先按 Ctrl+a，松开后按以下按键：

  • d：detach 当前对话
  • k：kill 当前对话

tmux

开启新 session（并命名）

显示所有session

使用 session 编号或名称接入

使用 session 编号或名称 kill

使用 session 编号或名称切换

重命名 session

Ctrl+b 系列：先按 Ctrl+b，松开后按以下按键：

  • s：选择需要跳转的 session 会话

  • $：重命名当前会话

  • d：断开当前 session

  • c：在当前 session 中多加一个 window

  • w：在一个 session 中的多个 window 中作出选择

  • x：关闭当前 session 中的当前 window

  • [：进入 tmux 翻屏模式, 实现上下翻页；进入翻屏模式后 PgUp PgDn 实现上下翻页（mac 可以用 fn + ↑ ↓ 实现上下翻页）；q 退出翻屏模式

  • !：关闭一个 session 中所有窗口

  • %：将当前窗口分成左右两分

  • "：将当前窗口分成上下两分

  • 方向键 ：让光标在不同的窗口中跳转

  • 按住 Ctrl+b 不放，同时按住方向键，可以调节光标所在窗口的大小

pigz

gzip 的并行实现。

  • 压缩单个文件

    加上-k选项保留原始文件，会在当前工作目录获得压缩后的your_file_name.gz 文件。

  • 解压单个文件

    同样，如果需要保留.gz文件，记得加上-k选项。

  • 压缩文件夹

    pigz 没有压缩文件夹的选项，只能压缩单个文件。若想压缩文件夹，可以结合 tar 使用：

  • 解压文件夹

常用参数：

  • -0 ~ -9 压缩等级，数字越大压缩率越高，速度越慢，默认为6
  • -k --keep 压缩后不删除原始文件
  • -l --list 列出压缩输入的内容
  • -K --zip Compress to PKWare zip (.zip) single entry format
  • -d --decompress 解压缩输入
  • -p --processes n 使用 n 核处理，默认为使用所有CPU核心
