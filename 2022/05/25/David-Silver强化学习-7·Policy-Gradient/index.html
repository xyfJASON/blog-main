[David Silver强化学习]7·Policy Gradient

\[ \newcommand{\calA}{\mathcal A} \newcommand{\calP}{\mathcal P} \newcommand{\calR}{\mathcal R} \newcommand{\calS}{\mathcal S} \newcommand{\bbP}{\mathbb P} \newcommand{\E}{\mathbb E} \newcommand{\bfw}{\mathbf w} \newcommand{\bfx}{\mathbf x} \]

1 Introduction

在以往的课程中，我们用线性函数或神经网络对 state-value function 或 action-value function 做估计： \[ V_\theta(s)\approx V^\pi(s)\quad Q_\theta(s,a)\approx Q^\pi(s,a) \] 然后根据贪心或 \(\epsilon\text{-greedy}\) 从 value function 中生成策略。

这节课我们直接参数化策略： \[ \pi_\theta(s,a)=\bbP[a\mid s,\theta] \] 对比 value-based 和 policy-based RL：

  • Value-based
    • 学习 value function
    • Policy 是隐式的
  • Policy-based
    • 没有 value function
    • 学习 policy
  • Actor-Critic
    • 学习 value function
    • 学习 policy
